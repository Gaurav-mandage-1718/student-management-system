# Student Management System

A simple and user-friendly Student Management System built using Spring Boot and Thymeleaf.  
This project helps users manage student records efficiently through a web-based interface.

## 🚀 Features

- Add Student
- View Students
- Update Student Details
- Delete Student
- Responsive User Interface
- CRUD Operations Implementation
- Server-side Rendering using Thymeleaf

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- Thymeleaf

### Backend
- Spring Boot
- Spring MVC
- Spring Data JPA

### Database
- MySQL

### Tools
- STS (Spring Tool Suite)
- Maven
- Git & GitHub


## ⚙️ CRUD Operations

This project performs complete CRUD operations:

| Operation | Description |
|----------|-------------|
| Create | Add new student records |
| Read | View all student records |
| Update | Edit existing student details |
| Delete | Remove student records |

## ⚙️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/Gaurav-mandage-1718/student-management-system.git
```

2. Open project in STS / IntelliJ IDEA

3. Configure MySQL database in `application.properties`

4. Run the Spring Boot application

5. Open browser and visit:

```bash
http://localhost:8080
```
